# Link Preview

> Auto-fetch URL meta info and generate rich preview cards

## Features
- Detects URLs in documents and fetches webpage info
- Parses Open Graph metadata (title, description, image)
- Generates styled preview cards
- Local cache for faster reloads

## Usage
Paste a URL in your Markdown document and save. A preview card will be generated automatically.

## Permissions
- **readContent**: Read document to detect URLs
- **writeContent**: Add preview cards
- **network**: Access the internet to fetch web data
- **log**: Record operation logs
