# 链接预览

> 自动获取 URL 的 meta 信息并生成预览卡片

## 功能特性
- 检测文档中的 URL 并获取网页信息
- 解析 Open Graph 元数据（标题、描述、图片）
- 生成精美的预览卡片
- 本地缓存加速加载

## 使用方法
在 Markdown 文档中粘贴 URL 后保存，预览卡片将自动生成。

## 所需权限
- **readContent**：读取文档以检测 URL
- **writeContent**：添加预览卡片
- **network**：访问互联网获取网页信息
- **log**：记录操作日志
