# 链接预览

> 自动获取 URL 的 meta 信息并生成预览卡片

## 功能特性

- **自动抓取**：检测文档中的 URL 并获取网页信息
- **Open Graph 支持**：解析 og:title/og:description/og:image
- **富文本卡片**：生成精美的预览卡片
- **智能缓存**：本地缓存减少网络请求

## 使用方法

在 Markdown 中粘贴 URL，保存后自动在文档末尾生成预览卡片。

## 权限

- readContent：读取文档内容
- writeContent：添加预览卡片
- network：访问网络获取网页信息
- log：记录日志
