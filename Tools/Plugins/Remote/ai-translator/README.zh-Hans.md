# AI 翻译器

> 使用 AI 服务翻译文本，支持多语言互译

## 功能特性
- AI 驱动的高质量翻译
- 支持中英文及其他主流语言
- 自动检测源语言
- 翻译统计记录

## 使用方法
用标记包裹要翻译的文本：
```
<!-- translate:en -->
要翻译的中文内容
<!-- /translate -->
```
保存文档后，译文将自动出现在下方。

## 所需权限
- **readContent**：读取待翻译文本
- **writeContent**：添加译文到文档
- **aiAccess**：调用 AI 翻译服务
- **log**：记录操作日志
