# AI 翻译器

> 使用 AI 服务自动翻译文本，支持多语言互译

## 📖 简介

AI 翻译器是一个远程插件，集成 ZhiYu 的 AI 服务，提供高质量的文本翻译功能。支持中英文互译、自动语言检测、翻译历史统计。

## ✨ 功能特性

- **AI 驱动**：使用先进的大语言模型提供准确翻译
- **多语言支持**：支持中文、英文及其他主流语言
- **自动检测**：智能识别源语言
- **标记语法**：使用简单的标记触发翻译
- **统计功能**：记录翻译次数和语言分布

## 🚀 使用方法

### 基础用法

使用标记语法指定要翻译的文本和目标语言：

**翻译为英文：**

```markdown
<!-- translate:en -->
这是一段中文文本。
<!-- /translate -->
```

保存后自动生成：

```markdown
<!-- translate:en -->
这是一段中文文本。

**译文 (en):**

This is a Chinese text.
<!-- /translate -->
```

**翻译为中文：**

```markdown
<!-- translate:zh -->
This is an English text.
<!-- /translate -->
```

### 支持的语言代码

| 代码 | 语言 |
|-----|------|
| `en` | 英文 |
| `zh` | 中文 |
| `ja` | 日文 |
| `ko` | 韩文 |
| `fr` | 法文 |
| `de` | 德文 |
| `es` | 西班牙文 |

### 命令列表

| 命令 ID | 命令名称 | 功能说明 |
|---------|---------|---------|
| `translate-to-en` | 翻译为英文 | 显示英文翻译用法 |
| `translate-to-zh` | 翻译为中文 | 显示中文翻译用法 |

### 工具栏按钮

- 🌐 **AI 翻译**：点击查看翻译统计

## ⚙️ 配置选项

本插件使用 ZhiYu 内置的 AI 服务，无需额外配置。

## 🔒 权限说明

本插件需要以下权限：

- **readContent**: 读取待翻译的文本
- **writeContent**: 添加译文到文档
- **aiAccess**: 调用 AI 服务进行翻译
- **log**: 记录日志信息

## 🌐 网络请求

⚠️ 本插件会将标记的文本发送到 ZhiYu AI 服务进行翻译。

### 隐私说明
- 仅发送标记区域内的文本
- 不会收集或存储您的文档
- 遵守 ZhiYu 隐私政策

## 💡 使用技巧

### 批量翻译

在文档中添加多个翻译标记，保存一次即可批量翻译：

```markdown
<!-- translate:en -->
第一段中文
<!-- /translate -->

<!-- translate:en -->
第二段中文
<!-- /translate -->
```

### 自定义语言

在标记中指定任何语言代码：

```markdown
<!-- translate:ja -->
Translate to Japanese
<!-- /translate -->
```
