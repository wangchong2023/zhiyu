# AI Translator

> Translate text using AI with multi-language support

## Features
- AI-powered high quality translations
- Supports Chinese, English and other major languages
- Auto-detects source language
- Tracks translation statistics

## Usage
Wrap the text to translate with markers:
```
<!-- translate:en -->
Your Chinese text here
<!-- /translate -->
```
Save the document and the translation will appear below.

## Permissions
- **readContent**: Read text for translation
- **writeContent**: Add translations to document
- **aiAccess**: Call AI translation service
- **log**: Record operation logs
