# AI Summary

> Extract key points and generate structured summaries

## Features
- AI-powered document summarization
- Extracts core points from long documents
- Generates concise structured summaries
- Supports multiple languages

## Usage
Save your document and the summary is generated automatically in post-processing.

## Permissions
- **readContent**: Read document for summarization
- **network**: Access AI service
- **log**: Record operation logs
