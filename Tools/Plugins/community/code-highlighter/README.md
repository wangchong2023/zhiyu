# Code Highlighter

> Add syntax highlighting to code blocks

## Features
- Syntax highlighting for code blocks
- Line numbers for better readability
- Supports multiple programming languages
- Lightweight processing

## Usage
Add code blocks to your document and the highlighting is applied on save.

## Permissions
- **readContent**: Read document for code detection
- **log**: Record operation logs
