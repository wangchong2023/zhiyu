## 📦 安装

### 从插件市场安装（推荐）

1. 打开 ZhiYu 应用
2. 进入「设置」→「插件中心」
3. 切换到「社区市场」标签
4. 搜索「插件名称」
5. 点击「安装」并启用

### 手动安装

1. 下载最新版本的 `.zyplugin` 文件
2. 打开 ZhiYu 应用
3. 进入「设置」→「插件中心」
4. 点击「加载本地插件」
5. 选择下载的 `.zyplugin` 文件
