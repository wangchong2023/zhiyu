# [本地] TOC 生成器

> 自动为 Markdown 文档生成目录索引

## 📖 简介

TOC 生成器是一个本地插件，能够自动扫描 Markdown 文档中的所有标题（H1-H6），并在文档顶部生成美观的目录索引。支持自动更新现有目录，无需手动维护。

## ✨ 功能特性

- **自动扫描标题**：智能识别 Markdown 中的所有 H1-H6 标题
- **生成目录索引**：自动生成层级缩进的目录结构
- **增量更新**：检测并更新已存在的目录，无需删除重建
- **统计功能**：记录目录生成次数和最后生成时间
- **离线可用**：无需网络权限，完全本地处理

## 📦 安装

### 手动安装（本地插件）

1. 下载 `toc-generator-local.zyplugin` 文件
2. 打开 ZhiYu 应用
3. 进入「设置」→「插件中心」→「我的插件」
4. 点击「加载本地插件」按钮
5. 选择下载的 `.zyplugin` 文件
6. 启用插件

## 🚀 使用方法

### 自动生成目录

插件会在文档保存时自动触发，无需手动操作：

1. 在 Markdown 文档中添加标题
2. 保存文档
3. 插件自动在第一个标题前生成目录

**生成效果：**

```markdown
<!-- TOC -->

- [第一章](#第一章)
  - [1.1 简介](#11-简介)
  - [1.2 安装](#12-安装)
- [第二章](#第二章)
  - [2.1 使用方法](#21-使用方法)

<!-- /TOC -->

# 第一章
## 1.1 简介
## 1.2 安装

# 第二章
## 2.1 使用方法
```

### 更新现有目录

如果文档中已存在 `<!-- TOC -->` 标记，插件会自动更新目录内容而不是重复生成。

### 命令列表

| 命令 ID | 命令名称 | 功能说明 |
|---------|---------|---------|
| `generate-toc` | 生成目录 | 手动触发目录生成 |

### 工具栏按钮

- 📋 **生成目录统计**：点击查看目录生成次数和最后生成时间

## ⚙️ 配置选项

本插件无需配置，开箱即用。

## 🔒 权限说明

本插件需要以下权限：

- **writeContent**: 修改文档内容以插入目录
- **log**: 记录日志信息

## 🌐 网络请求

✅ 本插件不需要网络权限，完全离线可用。

## 💡 使用技巧

### 自定义目录标记

插件使用 `<!-- TOC -->` 和 `<!-- /TOC -->` 作为目录标记，您可以：

- 手动在文档中预先添加这些标记来指定目录位置
- 删除这些标记后重新保存，插件会在默认位置重新生成

### 排除某些标题

如果不想某个标题出现在目录中，可以将其放在 `<!-- /TOC -->` 标记之后。

## 🐛 已知限制

- 不支持复杂的 Markdown 标题格式（如标题中包含代码或链接）
- 目录锚点仅支持字母、数字、中文和连字符
