# TOC Generator

> Auto-generate table of contents for Markdown documents

## Features
- Automatically scans H1-H6 headings and generates hierarchical TOC
- Updates existing TOC without duplication
- Tracks generation statistics
- Works completely offline

## Usage
Simply add headings to your Markdown document and save. The TOC is inserted automatically at the top.

## Permissions
- **writeContent**: Modify document to insert TOC
- **log**: Record operation logs
