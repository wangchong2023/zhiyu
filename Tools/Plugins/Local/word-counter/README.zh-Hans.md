# [本地] 字数统计

> 实时统计 Markdown 文档的字数、字符数和段落数

## 功能特性

- **精准统计**：智能识别中文字符和英文单词
- **语法过滤**：自动过滤 Markdown 语法标记
- **历史记录**：保存最近 10 次统计记录
- **离线可用**：无需网络权限

## 安装

1. 下载 `word-counter-local.zyplugin`
2. 打开智宇 App → 设置 → 插件中心 → 加载本地插件
3. 选择文件并启用

## 使用方法

保存文档后，点击工具栏的「字数统计」按钮查看详情。

## 权限

- readContent：读取文档内容
- log：记录日志
