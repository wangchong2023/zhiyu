# Smart Cleaner

> Auto-clean Markdown whitespace and formatting

## Features
- Merges 3+ consecutive blank lines into 2
- Removes trailing whitespace
- Normalizes line endings to LF
- Tracks cleaning statistics

## Usage
Save your document and whitespace is cleaned automatically.

## Permissions
- **writeContent**: Modify document content
- **log**: Record operation logs
