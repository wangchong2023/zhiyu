# 自动标签清洗插件 (Auto Tagger)

为 ZhiYu 用户设计的智能内容分类与格式格式化插件。能够识别和规范各种文本标记与任务代办项。

## 🌟 核心特性
- **任务标记清洗**：自动扫描文本并将各种 TODO、FIXME 等未完成关键词转换为标准的 Markdown 任务列表格式 (`- [ ]`)。
- **智能语义标签分配**：基于本地分词和规则引擎，分析文档主题并自动推荐关联标签，免去手动归类烦恼。
- **多平台无缝整合**：完全支持 iOS/macOS 快捷操作，在内容被保存或复制时可选择自动运行本插件进行格式整理。

## 🛠️ 使用指南
1. 插件激活后，将在编辑器顶栏生成一个闪烁小图标。
2. 在任意文档中输入带有 `TODO:` 或者 `FIXME:` 的文字，点击小图标即可自动运行清洗。
3. 您可以在设置页面自定义匹配的前缀正则以及清洗规则。
