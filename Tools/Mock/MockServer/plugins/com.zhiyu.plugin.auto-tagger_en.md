# Auto Tagger Plugin

An intelligent tagging and task formatting assistant for the ZhiYu platform.

## 🌟 Features
- **Task Keyword Cleaning**: Automatically scans your document and converts custom TODO/FIXME prefixes into standard Markdown task checkboxes (`- [ ]`).
- **Semantic Tag Suggestions**: Based on light local parsing, automatically suggests tags relevant to the text content.
- **Workflow Automation**: Option to automatically format text when copy-pasting or saving.

## 🛠️ Usage
1. Once enabled, an icon will appear in the editor toolbar.
2. Click the icon to clean any raw keywords into styled checkboxes.
3. Configure custom regex patterns in "Settings" -> "Auto Tagger".
