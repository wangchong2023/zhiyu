# Git Incremental Sync Plugin

A professional-grade version control and cloud synchronization plugin for the ZhiYu platform.

## 🌟 Features
- **Seamless Background Sync**: Automatically performs incremental commits and pulls on changes, keeping your remote repository up to date.
- **Intelligent Merge & Conflict Resolution**: Uses a custom three-way merge algorithm to resolve typical conflicts automatically.
- **Full Revision History**: View the historical modification curve of each document and roll back to any revision with a single click.
- **Resource Efficient**: Low CPU and disk I/O usage, highly optimized for mobile device battery life.

## 🛠️ Configuration
1. Go to "Settings" -> "Git Sync".
2. Input your Git Remote URL (supports HTTPS and SSH protocols).
3. Set your push interval (default is 5 minutes or upon manual save).
