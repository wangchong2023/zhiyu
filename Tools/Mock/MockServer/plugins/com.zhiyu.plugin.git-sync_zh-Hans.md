# Git 增量同步插件 (Git Incremental Sync)

ZhiYu 知识管理平台的专业级版本控制与云同步插件。支持自动触发同步，提供端到端数据一致性保障。

## 🌟 核心特性
- **无缝后台自动同步**：在本地修改后自动执行增量提交与拉取，始终保持远程库最新。
- **智能合并与自动消歧**：引入自定义三方合并机制，在检测到异地修改冲突时能够智能尝试自动消歧合并，最大化减少人工干预。
- **完整的版本链条回溯**：直观展示每一个页面的历史修改曲线，可直接一键回滚历史记录。
- **高效性能优化**：底层优化文件系统轮询，仅在修改时触发，电池功耗极其友好。

## 🛠️ 配置说明
1. 进入“插件设置” -> “Git 增量同步”。
2. 输入您的 Git Remote URL（支持 https 与 ssh 协议类型）。
3. 配置推送触发时间间隔（默认每 5 分钟或每次手动保存后）。
4. 在同步日志中可以查看每次的冲突自动消歧统计。
