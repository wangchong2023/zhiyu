## TaskCenterProtocol 编译错误修复
- **根因**: `TaskCenterProtocol: Sendable` 是 public 协议，但其参数类型 `TaskType`/`TaskStatus` 是 internal；`TaskCenter` 是非 final 类+`ObservableObject`，无法自动遵循 `Sendable`
- **修复**:
  1. `TaskType`/`TaskStatus` 改为 `public`（含计算属性和 `==` 运算符）
  2. `TaskCenter` 改为 `final class` + `@unchecked Sendable`
  3. 添加协议签名精确匹配方法: `updateTask(_:status:)` 和 `completeTask(id:)`（不带额外默认参数）
  4. `AppStore.saveToDisk()` 添加 `public`
  5. `SyncCoordinatorDelegate` 移除 `: Sendable`（@MainActor 隔离已足够）
