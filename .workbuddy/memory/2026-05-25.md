## 文档注释自动补齐
- **问题**: pre-commit hook (`generate_doc_comments.py --check`) 报 700 个非私有函数缺失 `///` 文档注释，阻塞 git commit
- **方案**: 编写 `Tools/autofill_doc_comments.py` 自动生成中文文档注释
  - 基于函数名驼峰拆词，动词前缀匹配 VERB_MAP (200+ 动词→中文映射)
  - 参数名推断中文标签，返回类型推断说明
  - 特殊模式处理：SwiftUI body、L10n tr/trf、协议委托回调、Equatable == 运算符
  - 首次运行 bug：`///` 前缀缺失（`f"{indent}{cl}\n"` → `f"{indent}/// {cl}\n"`），git checkout 还原后修复
- **结果**: 178 个文件 +697 条自动注释 + 3 条手动补齐（`static func ==` 运算符），总计 700 条
- **验证**: hook 通过 (0 missing)、iOS 编译通过 (BUILD SUCCEEDED)、git commit 成功 (57249f4)
