## ✅ 插件已复制到模拟器

插件位置：`Documents/Plugins/`

## 📱 测试步骤

### 1. 重启应用
在模拟器中完全退出并重新打开 ZhiYu 应用。

### 2. 进入插件中心
- 打开「设置」
- 点击「插件中心」
- 切换到「我的插件」标签

### 3. 验证插件加载
应该能看到 3 个插件：
- ✅ [本地] TOC 生成器
- ✅ [本地] 字数统计  
- ✅ [本地] 智能清洗器

---

## 🧪 功能测试

### 测试 1: TOC 生成器

1. 创建新笔记或打开已有笔记
2. 输入以下内容：
```markdown
# 第一章

## 1.1 简介

## 1.2 安装

# 第二章

## 2.1 使用方法
```
3. 保存文档
4. **期望结果**: 自动在文档顶部生成目录

### 测试 2: 字数统计

1. 打开任意笔记
2. 输入一些中英文混合内容
3. 保存文档
4. 点击工具栏的「字数统计」按钮
5. **期望结果**: 显示字数、字符数统计信息

### 测试 3: 智能清洗器

1. 创建新笔记
2. 输入内容，故意添加多个空行：
```markdown
内容1


内容2


内容3
```
3. 保存文档
4. **期望结果**: 自动合并多余空行，保留适当间距

## 🐛 故障排查

### 问题：插件未显示

**可能原因：**
1. 应用未重启
2. 插件文件格式错误
3. 安全模式已开启

**解决方法：**
```bash
# 1. 验证插件文件存在
ls -lh ~/Library/Developer/CoreSimulator/Devices/9ABEC5B9-E952-422A-A0AB-E2B785C1B36C/data/Containers/Data/Application/*/Documents/Plugins/

# 2. 检查插件完整性
python3 Tools/validate_plugin.py Tools/Plugins/Local/toc-generator-local.zyplugin

# 3. 查看应用日志
xcrun simctl spawn booted log stream --predicate 'subsystem == "com.zhiyu.app"' --level debug
```

### 问题：插件加载失败

**检查日志中的错误信息：**
- 权限问题
- manifest.json 格式错误
- index.js 语法错误
- 沙盒安全限制
