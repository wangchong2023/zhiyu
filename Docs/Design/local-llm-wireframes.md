## 7. 设计规范摘要

### 7.1 颜色语义

| 颜色 | 用途 | 示例 |
|------|------|------|
| 🟢 绿色 | 成功、就绪、健康 | 模型就绪、服务器在线 |
| 🟡 黄色/橙色 | 警告、临界状态 | 运存警告、低电量提示 |
| 🔴 红色 | 错误、限制、危险 | 内存不足、下载失败 |
| 🔵 蓝色 | 主操作、激活态 | 下载按钮、已激活模型 |
| ⚫ 灰色 | 次要操作、禁用态 | 取消按钮、不可用功能 |

### 7.2 交互反馈

- **点击反馈**: 0.1s 缩放动画 (scale: 0.95)
- **加载反馈**: Skeleton UI 或进度条
- **成功反馈**: 绿色闪烁 + Haptic
- **错误反馈**: 红色抖动 + Haptic
- **长按反馈**: 0.5s 后显示菜单

### 7.3 动画时序

- **页面切换**: 300ms slide
- **模态弹出**: 400ms spring
- **列表项展开**: 300ms ease-in-out
- **进度更新**: 线性（无动画）
- **状态变化**: 200ms fade
