## дәҢгҖҒеүҚз«ҜеҲҶеұӮжһ¶жһ„ (SwiftUI)

### 2.1 иЎЁзҺ°еұӮ Presentation Layer

| и§Ҷеӣҫз»„д»¶ | еҠҹиғҪжҸҸиҝ° |
|---------|---------|
| `LoginView` | зҷ»еҪ•жіЁеҶҢйЎөйқўпјҢеҢ…еҗ«OAuthзҷ»еҪ•жҢүй’®гҖҒжүӢжңәйӘҢиҜҒз Ғзҷ»еҪ• |
| `NotebookHubView` | з¬”и®°жң¬е·ҘдҪңеҸ°пјҢеұ•зӨәжүҖжңүз¬”и®°жң¬еҚЎзүҮ |
| `NotebookView` | з¬”и®°жң¬еҶ…е®№йЎөпјҢз®ЎзҗҶwikiгҖҒе°ҸзЁӢеәҸгҖҒAIеҠ©жүӢ |
| `SettingsView` | дёӘдәәи®ҫзҪ®йЎөпјҢиҙҰжҲ·з®ЎзҗҶгҖҒеҒҸеҘҪи®ҫзҪ® |

### 2.2 и§ҶеӣҫжЁЎеһӢеұӮ ViewModel Layer

|----------|-----|

### 2.3 жңҚеҠЎеұӮ Service Layer

| Service | иҒҢиҙЈ |
|---------|-----|
| `AuthService` | з»ҹдёҖи®ӨиҜҒжҺҘеҸЈпјҢе°ҒиЈ…зҷ»еҪ•/жіЁеҶҢ/зҷ»еҮәйҖ»иҫ‘ |

## дёүгҖҒеҗҺз«ҜеҲҶеұӮжһ¶жһ„ (Node.js/Express жҲ– Go)

### 3.1 зҪ‘е…іеұӮ Gateway Layer

| з»„д»¶ | иҒҢиҙЈ |
|-----|-----|
| `Kong / Nginx` | иҙҹиҪҪеқҮиЎЎгҖҒSSLз»Ҳз«ҜгҖҒиҜ·жұӮж—Ҙеҝ— |

### 3.2 жңҚеҠЎеұӮ Service Layer

| Service | иҒҢиҙЈ |
|---------|-----|
| `Auth Service` | OAuthи®ӨиҜҒжөҒзЁӢгҖҒTokenзӯҫеҸ‘дёҺеҲ·ж–°гҖҒзӯ–з•ҘжЁЎејҸеӨ„зҗҶеҗ„е№іеҸ°зҷ»еҪ• |
| `User Service` | з”ЁжҲ·жіЁеҶҢгҖҒдҝЎжҒҜжӣҙж–°гҖҒиә«д»ҪиҒ”еҗҲз®ЎзҗҶ |
| `Notebook Service` | з¬”и®°жң¬CRUDгҖҒеҲҶдә«дёҺеҚҸдҪң |

### 3.3 дёҡеҠЎйҖ»иҫ‘еұӮ Business Logic Layer

| з»„д»¶ | иҒҢиҙЈ |
|-----|-----|
| `OAuth Strategy` | зӯ–з•ҘжЁЎејҸе®һзҺ°пјҢж”ҜжҢҒеҫ®дҝЎ/QQ/Google/Appleеҗ„е№іеҸ°OAuth |
| `SMS Validator` | зҹӯдҝЎйӘҢиҜҒз Ғз”ҹжҲҗгҖҒж ЎйӘҢгҖҒ60з§’йҳІеҲ· |
| `Permission Service` | RBACжқғйҷҗжҺ§еҲ¶пјҢи§’иүІдёҺиө„жәҗз»‘е®ҡ |

### 3.4 ж•°жҚ®и®ҝй—®еұӮ Data Access Layer

| з»„д»¶ | иҒҢиҙЈ |
|-----|-----|
| `User Repository` | з”ЁжҲ·ж•°жҚ®CRUDж“ҚдҪң |
| `Notebook Repository` | з¬”и®°жң¬ж•°жҚ®CRUDж“ҚдҪң |

| `ORM (Prisma/TypeORM)` | ж•°жҚ®еә“ORMжҳ е°„ |
