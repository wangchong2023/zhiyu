# 欢迎来到智宇 (ZhiYu) 👋

探索真正的**AI 原生知识管理**魔法。在这里，您的每一篇笔记都会“思考”。

## 🌟 什么是智宇？

智宇不仅仅是一个精美的 Markdown 编辑器，它是一个内置了 **RAG (检索增强生成)** 引擎的私人数字大脑。这意味着：
- **永远不会遗忘**：把长篇文献、文章或灵感抛给智宇，系统会自动将它们转化为高维语义向量，永久铭记。
- **与知识对话**：您再也无需苦苦翻阅海量文件夹。点击界面上的 AI 对话面板，直接询问“总结这篇文档的核心”、“帮我提炼上周的研究报告”，AI 将像您的助理一样，从海量知识中瞬时提取并带引用总结。

## 🚀 您的魔法第一步

1. **随心记录**：修改这篇文档，或者在侧边栏新建一个页面，体验丝滑的写作感。
2. **连接孤岛**：使用 `[[页面名称]]` 语法可以轻松创建双向链接，观察左侧图谱的实时生长。
3. **点石成金**：留意页面底部的 **引导式 AI 浮窗**，立刻对这篇文档发起第一次知识对话提问！
